# Code of Conduct

This project follows the [Ansible Community Code of Conduct][ansible-coc].
Anyone participating in the project — issues, pull requests, discussions —
is expected to abide by it.

To report abusive, harassing, or otherwise unacceptable behaviour, contact
the project maintainer (see `galaxy.yml`) or the Ansible Diversity working
group as described in the link above.

[ansible-coc]: https://docs.ansible.com/ansible/latest/community/code_of_conduct.html
